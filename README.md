TPO para Codo a Codo - Python Full Stack (2022) comisión #22069

Las consignas eran:

1) Poseer al menos 3 páginas html, siendo su temática libre. En caso de ser un Sitio Web estilo
One Page, el mismo deberá tener al menos 3 secciones.
   **En este caso tenemos 3 páginas: index.html, canciones.html y formulario-contacto.html.**
   
2) Utilizar etiquetas semánticas para estructurar el sitio.
  **Se utilizan etiquetas semánticas en todos los lugares posibles.**

3) Contener un formulario de contacto con validación realizada mediante Javascript.
  **La página formulario-contacto.html posee un formulario de contacto, valga la redundancia, que le pide al usuario que introduzca: nombre, email, asunto, y mensaje.
  A su vez, se diseñó una función en myScript-formulario.js que verifica los datos que introduce el usuario y le alertan de acuerdo a si son válidos o no.** 
  
4) Utilizar al menos un Iframe, íconos de FontAwesome, y fuentes locales o bien de Google
Fonts.
  **El iframe está siendo utilizado en la página canciones.html. También se utiliza un icono de FontAwesome en index.html (el icono del mail). Por último, todas las páginas hacen uso de la font Nunito, sacada de Google fonts.**
  
5) Tener al menos una página totalmente responsive con 3 puntos de corte para 3 tamaños de
dispositivos distintos.
  **La página index.html posee 3 puntos de corte: uno para mobile, uno para tablets, y otro para desktops.**
  
6) Utilizar al menos una animación, transformación o una transición (se deberá indicar dónde
fue incorporada la animación).
  **La animación se encuentra en la página index.html (el icono del mail hace un slide in desde la derecha).**
  
7) Poseer una estructura HTML maquetada con Flexbox y/o Grid.
  **index.html está completamente maquetada con una combinación de Flexbox y Grid. Las otras dos páginas utilizan Flexbox cuando fuere necesario.**
